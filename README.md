# Ma Wishlist 🎁

Une page web simple et élégante pour partager ma liste de souhaits.

## 📋 Description

Cette page HTML présente une liste de cadeaux que j'aimerais recevoir, avec un design moderne et responsive.

## 🚀 Fonctionnalités

- Design responsive et moderne
- Interface utilisateur intuitive
- Cartes de produits avec images
- Liens directs vers les produits

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- Design responsive

## 📱 Aperçu

La page est accessible via GitHub Pages et présente une grille de cartes de produits avec des images, descriptions et liens.
